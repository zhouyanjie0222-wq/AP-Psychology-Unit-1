# AP 心理 Unit 1 抽查网页（像素风）

## 使用方法

- 直接用浏览器打开 `index.html` 即可（无需安装任何东西）。
- 点击一次：随机抽取一个 Unit 1 术语  
- 再点一次：显示该术语解释  
- 再点一次：进入下一题  

快捷键：

- `Space` / `Enter`：等同于点击主按钮

## 题库来源

- 术语清单来自你提供的 `Unit1-Biological.pdf` 的 vocab 列表
- 每条解释按 AP Psych 常见课堂口径整理，便于抽查

## 位置

- 文件夹：`ap-psych-unit1-checker/`
- 入口：`ap-psych-unit1-checker/index.html`
